## Tips for Efficient Research

1. **Use Property Finder Filters**: Select "Rent", choose neighborhood, filter by property type
2. **Look for Patterns**: After 5-6 listings, you'll see typical ranges
3. **Note Outliers**: Exceptionally high/low prices (penthouse vs. older building)
4. **Check Map View**: See proximity to metro, beach, major landmarks
5. **Read Descriptions**: Amenities, building age, community features
